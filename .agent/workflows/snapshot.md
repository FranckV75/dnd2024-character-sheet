# Workflow Snapshot Visuel

## Étapes

1. **Ouvrir le navigateur** sur `index.html` à 1024px de largeur

2. **Capturer chaque onglet** :
   - Tab 1 : Caractéristiques & Combat
   - Tab 2 : Compétences & Histoire
   - Tab 3 : Magie, Équipement & Trésors

3. **Sauvegarder les screenshots** avec format :
   - `snapshot_tab1_[date].png`
   - `snapshot_tab2_[date].png`
   - `snapshot_tab3_[date].png`

4. **Créer un mini-rapport** listant :
   - Screenshots capturés
   - Version/commit actuel
   - Notes visuelles si nécessaire

## Utilisation

Exécuter ce workflow :
- Avant une refactorisation majeure
- Après un audit réussi
- Pour documenter une nouvelle fonctionnalité
